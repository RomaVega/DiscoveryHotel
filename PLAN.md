## Ключевые принципы для "wow-эффекта"

1. **Видео в hero** — первое впечатление решает всё
2. **Реальные фото** — никаких stock, только настоящий отель
3. **Плавные анимации** — FadeIn, stagger, hover — создают ощущение премиальности
4. **Много воздуха** — щедрые отступы (py-24, py-32), не загромождать
5. **Единый стиль** — brand-teal акценты, Cormorant для заголовков, Inter для текста
6. **Мобильная версия** — заказчик скорее всего откроет на телефоне первым делом
